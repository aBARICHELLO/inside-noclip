# inside-noclip
NOCLIP Cheat Engine table with Lua scripts for Playdead's INSIDE game.

# Usage
1. Open the game.
2. File > Open File. Select INSIDE.CT.
3. File > Open Process. Select the game "INSIDE"
4. Mono > Activate Mono Features.
5. Enjoy.

# HotKeys
SOON tm.
